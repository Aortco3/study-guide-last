HTML

  • El head elemento contiene informaciónmetadatos que ayuda al navegador a mostrar la página correctamente.
  • Body elemento contiene el contenido que se muestra al usuario Todos los elementos que queremos que el usuario vea e interactúe deben colocarse dentro de este elemento.
  • Dentro del Head podemos usar las siguientes etiquetas de metadatos.
  • El title elemento define el título de una página web Este elemento es obligatorio y es extremadamente importante para la optimización de motores de búsqueda (SEO).
  • El style elemento define los estilos CSS asociados con un sitio web.
  • El base elemento define la URL base de una página web.
  • El link elemento conecta un recurso externo al documento HTML.
  • El meta elemento define metadatos como el conjunto de caracteres, la descripción, las palabras clave, el autor y la ventana gráfica.
  • El script elemento incrusta secuencias de comandos del lado del cliente en el documento HTML.

CSS

  • La margin propiedad actúa como un margen en un documento de procesamiento de texto Una margin propiedad indica cuánto espacio queremos alrededor delexterior de un elemento.
  • La padding propiedad agrega espacio alrededor delcontenido dentro de un elemento.
  • Cuando creamos un archivo CSS externo, debemos vincularlo al archivo HTML ejemplo link rel="stylesheet" href="./assets/style.css"
  • Card designa esto como un selector de clase. Cualquier elemento HTML asignado a la clase card obtendrá estas asignaciones de propiedades del CSS width, margin, border, box-shadow

Git

  • git init: lo usamos para determinar la carpeta en la que vamos a trabajar.
  • git checkout -b branch-name: creates a new branch and switches to it
  • git status lo usamos para saber si tenemos un archivo añadido o borrado en nuestro proyecto, para saber en la rama en la que estamos y si tenemos commits.
  • git add: es para añadir un archivo a nuestra rama seguidamente ponemos entre comillas el nombre de nuestro archivo o poner un punto para añadir todos los archios de nuestra carpeta.
  • git rm: lo usamos para borrar un archivo que hayamos añadido, para eliminarlo por completo de nuestra rama usamosgit rm --cached.
  • git commit: se usa para añadir un commit a nuestra rama, también podemos ponerle un -m seguidamente ponemos entre comillas nuestro ensaje.
  • git config: muestra configuraciones de git también podemos usar list para mostrar la configuración por defecto de nuestro git y si añadimos --show-origin inhales nos muestra las configuraciones guardadas y su ubicación.
  • git config --global user.name: cambia de manera global el nombre del usuario, seguidamente ponemos entre comillas nuestro nombre.
  • git config --global user.email: cambia de manera global el email del usuario, seguidamente ponemos entre comillas nuestro nombre.
  • git log: se usa para ver la historia de nuestros archivos, los commits, el usuario que lo cambió, cuando se realizaron los cambios etc. seguidamente ponemos el nombre de nuestro archivo.

JavaScript

  • CavaScript (JS) es un lenguaje de programación dinámico de alto nivel que permite a los usuarios interactuar con sitios web
  • Características básicas de JavaScript, como variables: es simplemente un contenedor con nombre que nos permite almacenar datos en nuestro código, En JavaScript, para declarar una variable, usamos var despues de var le damos un nombre a la variable.. Finalmente, usamos un operador de asignación =para asignar los datos específicos que queremos almacenar en nuestra variable Ejem: (var shape = "circle";)
  • Características básicas de JavaScript, como Cadenas Todas las cadenas están entre comillas. Tenga en cuenta que cada línea de código JavaScript termina con un punto y coma(;)
  • Características básicas de JavaScript, como Tipos de Datos
  • También veremos cómo se pueden usar las matrices para almacenar tipos de datos en JavaScript, y usaremos un for bucle para operar en cada elemento de una matriz
  • El console.log método se creó para facilitar a los desarrolladores la prueba de su código. Los desarrolladores lo usan a diario para depurar y probar el código que escriben
  • El flujo de control es el orden en que una computadora ejecuta el código en un script. En JavaScript, el código se ejecuta desde la primera declaración en la parte superior de la página hasta la última declaración en la parte inferior de la página.
  • If declaración, el código solo se ejecutará si la declaración cumple una condición o es verdadera.
  • El operador de igualdad estricta ( ===) verifica si dos valores son iguales y devuelve un resultado booleano true
  • Una matriz es una sola variable que se utiliza para contener un grupo de datos. Las matrices se utilizan normalmente para contener datos que están relacionados de alguna manera.
  • Un for bucle contiene las siguientes tres declaraciones importantes: (A) punto de partida de nuestro bucle (B) ejem (var x = 0); (B)condición . Siempre que el número en la variable x sea menor que la longitud de la matriz, el bucle seguirá ejecutándose ejem (x shapes.length;) (C) La declaración final es lo que permite que la matriz itere sobre cada elemento.
